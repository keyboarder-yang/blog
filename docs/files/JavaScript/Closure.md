#### 相关知识概要

> 定义：
> 	+ 一个函数和它周围状态的引用捆绑在一起的组合。
> 	+ 内部函数可以访问它们所在的外部函数中声明的所有局部变量、参数和声明的其他内部函数。当其中一个这样的内部函数在包含它们的外部函数之外被调用时，就会形成闭包。
> 	+ 闭包就是一个没有释放资源的栈区。
>
> 执行上下文：程序执行时，有自己的执行上下文，执行上下文分为两个阶段：
>
> + 创建阶段：
>   + **作用域链**：当前变量对象 + 所有父级变量对象 = 作用域链
>   + 变量对象：包括参数、变量、函数声明
>   + this
> + 执行阶段：变量赋值，函数引用等
